# SPTcl-IRAGN
**SPTcl-IRAGN** is a study designed to investigate the influence of infrared-bright active galactic nuclei (AGN) in the South Pole Telescope (SPT) cluster surveys.

[![Documentation Status](https://readthedocs.org/projects/sptcl-iragn/badge/?version=latest)](https://sptcl-iragn.readthedocs.io/en/latest/?badge=latest)

Documentation for core SPTcl-IRAGN modules can be found [here](https://sptcl-iragn.readthedocs.io/en/latest/)

This work was developed as part of Benjamin Floyd's PhD Dissertation which can be found on the [MOspace Dissertation Archive](https://hdl.handle.net/10355/101862).
